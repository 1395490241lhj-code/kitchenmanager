## 干烧鲫鱼

- id: `ex--497d6220`
- type: 烧鱼
- confidence: medium
- 食材: 鲫鱼
- 当前候选做法:
  1. 鲫鱼治净，两面煎（或炸）至定型。
  2. 锅留底油，爆香姜蒜（干烧可下豆瓣炒出红油）。
  3. 加料酒、酱油和适量汤，下鱼烧开转小火。
  4. 烧至入味、汤汁浓稠，大火收汁亮油起锅。
- 你的审核建议: 可直接采用（建议把「干烧可下豆瓣」改为必下豆瓣）
- 风险点:
  - 干烧讲究「汁少亮油」，豆瓣应是必备而非可选
  - 缺少打花刀、少许糖/泡椒等细节
- 建议修改版:
  1. 鲫鱼治净，两面打浅花刀，用盐、料酒抹匀，下油煎至两面金黄定型。
  2. 锅留底油，下郫县豆瓣、姜蒜末（可加泡椒、肉末）炒出红油。
  3. 加料酒、酱油、少许糖和半碗汤，下鱼小火干烧入味。
  4. 中途翻面，烧至汤汁将干、亮油，撒葱花起锅。

## 豆腐鲫鱼

- id: `ex--8aec747f`
- type: 豆腐鱼
- confidence: medium
- 食材: 鲫鱼、豆腐
- 当前候选做法:
  1. 鲫鱼治净，两面煎至微黄定型。
  2. 爆香姜蒜，加入适量热水或高汤烧开。
  3. 下豆腐同烧，小火煮至入味、汤色乳白。
  4. 调味收汁，撒葱花起锅。
- 你的审核建议: 可直接采用
- 风险点:
  - 奶白汤的关键是「煎透 + 热水大火滚」，候选已覆盖，整体稳
- 建议修改版:
  1. 鲫鱼治净，两面煎至微黄定型；豆腐切块。
  2. 爆香姜蒜，加足量热水大火烧开，滚出奶白鱼汤。
  3. 下豆腐同煮，小火炖至入味、汤色乳白。
  4. 加盐调味，撒葱花（喜辣可加少许泡椒）起锅。

## 干煸鳝鱼

- id: `ex--efa871b1`
- type: 干煸
- confidence: medium
- 食材: 鳝鱼
- 当前候选做法:
  1. 鳝鱼处理干净，改刀成条或段。
  2. 锅下少油，中火把鳝鱼煸炒至水分收干、表面微皱。
  3. 下姜蒜、干辣椒、花椒（荤料可加豆瓣）炒香。
  4. 调味后继续翻炒至干香入味，起锅。
- 你的审核建议: 可直接采用（可补「芹菜/蒜苗」配菜）
- 风险点:
  - 传统干煸鳝鱼常配芹菜或蒜苗，数据无配菜信息，未体现
- 建议修改版:
  1. 鳝鱼治净去骨，切段沥干水分。
  2. 锅下油，中火把鳝鱼段煸炒至水汽收干、边缘微焦酥香。
  3. 下姜蒜、干辣椒、花椒、豆瓣炒香，可加芹菜或蒜苗同煸。
  4. 烹少许料酒、酱油，翻炒至干香入味，起锅。

## 魔芋烧鸭

- id: `ex--56a6ad20`
- type: 烧菜/红烧
- confidence: medium
- 食材: 鸭肉、魔芋
- 当前候选做法:
  1. 鸭肉、魔芋洗净切块，肉类先焯水或煸炒定型。
  2. 起锅烧油，爆香姜蒜（可下豆瓣、干辣椒、花椒）。
  3. 下主料煸炒上色，烹入料酒、酱油。
  4. 加热水没过食材，烧开转小火炖至软糯入味。
  5. 调味后大火收汁，起锅装盘。
- 你的审核建议: 可直接采用（建议补「魔芋焯水去碱味」）
- 风险点:
  - 魔芋需先焯水去碱味，否则有涩味
- 建议修改版:
  1. 魔芋切条焯水去碱味；鸭肉斩块焯水沥干。
  2. 起锅烧油，下鸭块煸炒出油，加姜蒜、郫县豆瓣炒香上色。
  3. 烹料酒、酱油，加热水没过，烧开转小火焖约 20 分钟。
  4. 下魔芋同烧至入味，大火收汁，撒葱花起锅。

## 泡菜鱼

- id: `ex--a276fea5`
- type: 泡菜鱼
- confidence: medium
- 食材: 鲜鱼、泡菜
- 当前候选做法:
  1. 鲜鱼治净切块，用盐、料酒码味。
  2. 锅下油，炒香泡菜、泡椒、姜蒜出香出色。
  3. 加适量汤或清水烧开，调味。
  4. 下鱼块煮至刚熟入味，连汤起锅。
- 你的审核建议: 可直接采用
- 风险点:
  - 鱼块可加少许淀粉码味更嫩，候选未提
- 建议修改版:
  1. 鲜鱼治净切块，用盐、料酒、少许淀粉码味。
  2. 锅下油，炒香泡菜、泡椒、姜蒜，出酸香红色。
  3. 加适量汤或清水烧开，调成酸辣汤底。
  4. 下鱼块中小火煮至刚熟嫩滑，连汤起锅，撒葱花。

## 姜爆鸭丝

- id: `ex--206c805b`
- type: 炒菜
- confidence: medium
- 食材: 鸭肉
- 当前候选做法:
  1. 鸭肉切丝（片），加盐、料酒、水淀粉码味上浆。
  2. 热锅下油，下葱姜蒜爆香（姜爆类多放姜丝）。
  3. 下主料大火快炒至变色断生。
  4. 加盐、生抽等调味，翻炒均匀起锅。
- 你的审核建议: 可直接采用（姜爆讲究「爆」，可弱化厚浆）
- 风险点:
  - 姜爆类以姜丝为主味，厚浆会糊锅，建议略码味即可
- 建议修改版:
  1. 鸭肉切细丝，加盐、料酒略码味（姜爆不必厚浆）。
  2. 热锅多油，下鸭丝大火爆炒至散籽变色。
  3. 下大量姜丝（少许泡椒、青椒丝）爆香。
  4. 烹料酒、生抽快速翻匀，断生即起锅，保持鲜辣爽口。

## 碎米豆腐

- id: `ex--c410dfe1`
- type: 豆腐菜
- confidence: medium
- 食材: 豆腐
- 当前候选做法:
  1. 豆腐改刀成丁或块，入淡盐水焯一下沥干。
  2. 锅下油，爆香姜蒜（可下肉末或豆瓣炒香）。
  3. 加少量汤下豆腐轻烧，调味入味。
  4. 勾薄芡收汁，撒葱花起锅。
- 你的审核建议: 需要小修
- 风险点:
  - 「碎米豆腐」核心是肉末（碎米）+ 豆豉的咸鲜微辣，候选写得太通用
- 建议修改版:
  1. 豆腐切小丁，入淡盐水焯一下沥干。
  2. 锅下油，下肉末炒散出油，加姜蒜末、豆瓣、豆豉炒香出红油。
  3. 下豆腐丁，加少量汤轻烧入味。
  4. 勾薄芡收汁，撒花椒面和葱花起锅。

## 辣子鱼

- id: `ex--c6960e2c`
- type: 辣子/陈皮
- confidence: medium
- 食材: 鲜鱼
- 当前候选做法:
  1. 鲜鱼治净改刀，加盐、料酒码味。
  2. 下油锅炸（或煎）至外表酥香、定型，捞出。
  3. 锅留底油，爆香大量干辣椒节和花椒（陈皮可同下）。

  - 鱼块宜拍粉炸更酥
- 建议修改版:
  1. 鱼治净切块，加盐、料酒、姜葱码味，拍薄淀粉。
  2. 下油锅炸至外酥定型，捞出。
  3. 锅留底油，下大量干辣椒节、花椒煸出煳辣香。
  4. 倒入鱼块快速翻炒裹味，撒熟芝麻、葱花起锅。

## 鱼香茄饼

- id: `ex--73715b26`
- type: 鱼香
- confidence: medium
- 食材: 茄子
- 当前候选做法:
  1. 茄子改刀（茄饼可夹入肉馅、挂糊炸至金黄）。
  2. 用糖、醋、酱油、水淀粉兑成鱼香味汁。
  3. 锅下油，爆香泡椒、姜蒜末出香出色。
  4. 下主料略烧，烹入鱼香汁收汁亮油，撒葱花起锅。
- 你的审核建议: 需要小修
- 风险点:
  - 茄饼通常是「浇汁 / 裹汁」而非把脆茄饼回锅久烧，「略烧」措辞稍偏
- 建议修改版:
  1. 茄子切两刀不断的夹刀片，夹入调好味的肉馅。
  2. 茄饼挂蛋糊，下油锅炸至两面金黄熟透，摆盘。
  3. 用糖、醋、酱油、水淀粉兑成鱼香汁。
  4. 锅下油爆香泡椒、姜蒜末，烹入鱼香汁熬浓，浇在茄饼上，撒葱花。

## 糯米圆子

- id: `ex--177d2fcb`
- type: 甜点/圆子
- confidence: medium
- 食材: 糯米
- 当前候选做法:
  1. 糯米提前浸泡，沥干（或磨浆和成糯米团）。
  2. 搓成大小均匀的圆子，可包入豆沙或白糖馅。
  3. 下沸水煮至圆子浮起、熟透软糯（或上笼蒸熟）。
  4. 捞出按口味裹糖、淋醪糟或撒桂花即可。
- 你的审核建议: 需要小修（先确认是醪糟版还是包馅版）
- 风险点:
  - 四川常见「醪糟糯米圆子 / 红糖糯米圆子」，候选偏笼统，需确定口味方向
- 建议修改版:
  1. 糯米粉加温水和成软面团（或泡糯米打浆）。
  2. 搓成小圆子（可包红糖或豆沙馅）。
  3. 下沸水煮至浮起、熟透软糯。
  4. 可下入醪糟红糖汤同煮，或捞出裹黄豆粉 / 白糖食用。

## 葱酥鱼

- id: `ex--baed7d9b`
- type: 烧鱼（识别偏差）
- confidence: medium
- 食材: 鲜鱼、葱
- 当前候选做法:
  1. 鲜鱼治净，两面煎（或炸）至定型。
  2. 锅留底油，爆香姜蒜（干烧可下豆瓣炒出红油）。
  3. 加料酒、酱油和适量汤，下鱼烧开转小火。
  4. 烧至入味、汤汁浓稠，大火收汁亮油起锅。
- 你的审核建议: 建议重写
- 风险点:
  - **菜型识别偏差**：葱酥鱼是「小鱼炸酥 + 大量葱姜垫底 + 糖醋慢㸆至骨酥」的酥鱼，候选误用了干烧 / 豆瓣方向
  - 重点是「葱香」「骨酥」，不是收汁亮油的麻辣干烧
- 建议修改版:
  1. 小鲫鱼治净沥干，下油锅炸至外表酥脆，捞出。
  2. 锅垫一层大葱段、姜片，码上炸好的鱼。
  3. 加酱油、醋、糖、料酒和没过鱼的汤，小火慢㸆。
  4. 㸆至汤汁将干、葱香入味、鱼骨酥软，连葱装盘。
