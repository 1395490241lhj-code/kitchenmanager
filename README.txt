
# kitchenmanager v19（在线编辑版，无需改底层文件）
- 新增 `#editor` 页面：在手机/电脑上直接添加/修改/隐藏菜谱，数据保存到本地 `localStorage` 的“补丁（overlay）”中。
- 任何修改**不会**改动仓库里的 `data/sichuan-recipes.json` 文件。
- 支持导入/导出补丁（JSON），方便在多设备间同步。
- 其它页面（菜谱/推荐/库存/购物清单）都会使用“基础数据 + 你的补丁”的合并结果。

部署：把整个压缩包解压后，全部文件上传到仓库根目录；强制刷新 (?v=19)。若看到旧页面，打开 sw-reset.html 清缓存。
